# 탄단지 🍎 - 음식 영양 정보 검색 웹사이트

음식 이름을 입력하면 열량, 칼로리, 영양소 정보를 제공하고 AI와 의학적 전문가 평가를 해주는 웹 애플리케이션입니다.

## ✨ 주요 기능

- 🔍 **범용 음식 검색**: 어떤 음식이든 입력하면 카테고리 기반으로 자동 인식
- 📊 **정확한 영양소 정보**: 칼로리, 탄수화물, 단백질, 지방, 섬유질, 당류 제공
- 🤖 **AI 건강 평가**: 현재 시간대와 어제 식사 기록을 분석한 맞춤 평가
- 🏥 **의학적 종합 평가**: 영양학 전문가 기준 상세 분석 (A~F 등급)
- 📝 **식사 기록 관리**: 오늘/어제 먹은 음식 기록 및 칼로리 추적
- ⚠️ **부족한 영양소 분석**: 단백질, 섬유질 등 부족한 영양소 알림
- 💡 **건강 영향 분석**: 단기/장기 건강 영향 설명
- 🎯 **전문가 권장사항**: 구체적인 식이 조언 제공

## 🚀 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173`로 접속하세요.

### 빌드

```bash
npm run build
```

## 📋 주요 기능 상세

### AI 건강 평가
- 현재 시간대별 칼로리 권장량 분석
- 하루 총 칼로리 대비 평가
- 어제 식사 기록과 비교 분석
- 영양소 균형 평가 (지방, 당분, 단백질, 섬유질)

### 의학적 종합 평가
- **종합 등급**: A(우수) ~ F(매우 부적절)
- **상세 분석**: 구체적인 수치와 기준 제시
- **건강 영향**: 단기/장기 영향 설명
- **위험 요인**: 고칼로리, 고지방, 당분 과다 등
- **전문가 권장사항**: 구체적인 식이 조언

### 범용 음식 인식
데이터베이스에 없는 음식도 키워드 분석으로 자동 인식:
- 과일류, 채소류, 육류, 해산물, 면류, 찌개류 등 카테고리 자동 분류
- 카테고리별 평균 영양소 제공

## 📝 지원 음식 (데이터베이스)

70개 이상의 음식 데이터가 포함되어 있으며, 카테고리 기반으로 확장 가능합니다.

## 📖 사용 방법

1. **음식 검색**: 검색창에 음식 이름 입력 (예: "치킨", "피자", "사과", "샐러드")
2. **영양 정보 확인**: 칼로리 및 상세 영양소 정보 확인
3. **AI 평가 확인**: 현재 시간과 식사 기록 기반 건강 평가
4. **의학적 평가 확인**: 전문가 수준의 종합 등급 및 상세 분석
5. **식사 기록**: "식사 기록" 버튼으로 오늘 먹은 음식 저장
6. **기록 관리**: "식사 기록 보기" 버튼으로 오늘/어제 기록 확인

## API 설정 (선택사항)

더 많은 음식 데이터를 사용하려면 Edamam Food Database API를 사용할 수 있습니다:

1. [Edamam](https://www.edamam.com/)에서 계정 생성
2. Food Database API 앱 생성
3. `.env` 파일 생성:

```
VITE_EDAMAM_APP_ID=your_app_id
VITE_EDAMAM_APP_KEY=your_app_key
```

API 키가 없어도 기본 제공되는 한국 음식 데이터로 테스트할 수 있습니다.
